# Обновлятор 1c linux

**Версия:** 1.0  
**App ID:** `io.github.kerim1c.updater1clinux`

Приложение для Linux для групповой работы с информационными базами 1С: проверка настроек, определение версии конфигурации, скачивание обновлений и подготовка к пакетному обновлению.

## Идея проекта

За основу взята идея и общий подход из приложения «Обновлятор-1С» для Windows:

https://helpme1s.ru/obnovlyator-1s-gruppovoe-paketnoe-obnovlenie-vsex-baz-za-odin-raz

Выражаю благодарность Милькину В.В., разработчику Windows-приложения, которое вдохновило на создание Linux-версии.

Проект создан в текущих реалиях активного использования Linux-систем как удобный инструмент для работы с базами 1С на Linux.

## Что нового в версии 1.1

- Исправлено скачивание обновлений для нескольких баз, отмеченных галочками.
- Исправлено определение обновлений для `Бухгалтерия предприятия КОРП`: теперь используется код `AccountingCorp`, а не `Accounting`.
- Добавлена защита файла списка баз 1С `ibases.v8i` от небезопасной перезаписи.
- Учтен фактический Linux-путь списка баз 1С: `~/.1C/1cestart/ibases.v8i`.
- Добавлен цветной структурированный отчет без поломки layout формы.
- Добавлен regression-test для защиты списка баз 1С.

## Возможности версии 1.1

- работа со списком баз 1С;
- группы баз;
- импорт списка баз;
- проверка параметров базы;
- автоматическая проверка релиза перед скачиванием обновлений;
- скачивание обновлений конфигураций;
- подготовка к обновлению баз на Linux;
- чистый Linux-интерфейс с App ID `io.github.kerim1c.updater1clinux`.
