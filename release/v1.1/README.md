# updater1c-linux 1.1

## Изменения

- Изменен формат имени `.cf`-бэкапов: `НаименованиеБазы_ТекущийРелиз_Дата.cf`.

- Исправлена защита списка баз 1С `ibases.v8i` от небезопасной перезаписи.
- Исправлено скачивание обновлений для нескольких отмеченных галками баз.
- Добавлен цветной структурированный отчет без поломки layout формы.
- Исправлено определение кода обновлений для `Бухгалтерия предприятия КОРП`: теперь используется `AccountingCorp`, а не `Accounting`.
- Исправлен вывод последовательности обновлений: вместо UUID отображаются версии релизов.

## Важно

Для обычной `Бухгалтерия предприятия` используется `Accounting`.
Для `Бухгалтерия предприятия КОРП` используется `AccountingCorp`.
